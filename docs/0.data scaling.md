For our models, the scaling is different for **inputs** and **outputs**.

**Inputs**
For the physical control variables, we used log transforms first:
- `E/N` was transformed to `log10(E/N)`.
- `Power` was transformed to `log10(power)`.

For gas composition features:
- We removed composition features that were constant in the training split.
- For PCA-input feature sets, the remaining composition features were standardized before PCA:  
  \[
  x' = \frac{x - \mu_{\text{train}}}{\sigma_{\text{train}}}
  \]
- PCA was then fit only on the training data, and validation/test were projected using that same scaler and PCA basis.
- For non-PCA feature sets, the models used the mole-fraction features plus `log10(E/N)` and `log10(power)`. For FFN models, these input features were also standardized internally so training was numerically stable.

**Outputs**
Both RATE CONST and SUPER RATE are nonnegative and span many orders of magnitude, so we trained and evaluated mainly in log space:
\[
y_{\log,j} = \log_{10}(y_j + \epsilon_j)
\]

Here, \(\epsilon_j\) is reaction-specific and estimated from the training split:
\[
\epsilon_j = \max(\frac{\min(y_j > 0)}{10}, 10^{-30})
\]

This prevents zeros from becoming \(-\infty\) after the log transform. After prediction, we convert back to original units by:
\[
\hat{y}_j = 10^{\hat{y}_{\log,j}} - \epsilon_j
\]

For latent-output models, we did one extra step:
1. transform outputs to `log10(y + epsilon)`;
2. apply PCA to the transformed output matrix;
3. train the model to predict PCA latent coefficients;
4. decode latent coefficients back to 204 outputs;
5. inverse-transform back to original values.

So the short presentation version is:

> Inputs were cleaned by removing constant gas-composition features, then physical controls `E/N` and power were log-transformed. For PCA and neural-network models, features were standardized using training-set statistics only. Outputs were transformed as `log10(y + epsilon)` because the rate coefficients are nonnegative, sparse, and span many orders of magnitude; predictions were converted back to original units for final evaluation.
